### Step 3:

using formula find distance
### Step 5:

### PROGRAM:
```python
#Program to find the distance between two points.
#Developed by: `kavya

import math
l1 = [4,2]
l2 = [10,6]
distance = math.sqrt(((l2[0]-l1[0])**2)+((l2[1]-l1[1])**2))
print("{:.2f}".format(distance))
```
